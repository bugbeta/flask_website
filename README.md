# flask_website
使用flask开发一个博客网站

## 功能

- 博客（已完成功能）

  1. 博客文章展示
  
  2. 创建文章
  
  3. 修改文章
  
  4. 增加文章评论
  
  5. 文章归档
   
- 个人主页（已完成功能）
  
  1. 个人简介展示
  
  2. 个人动态发布
   
  3. 个人动态展示
  
  4. 个人信息修改
